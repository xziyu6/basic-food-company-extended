# Basic Food Company Builds Other Grain Farms

This Victoria 3 mod allows the Basic Food Company to build all types of grain farms (wheat, rice, maize, and rye) via Industry Rights Charters.

## What This Mod Does

In the base game, the Basic Food Company can only build certain types of grain farms through Industry Rights Charters. This mod expands that capability by making all grain farm types available:

- Wheat Farms
- Rice Farms
- Maize Farms
- Rye Farms

## Installation

1. Download or clone this repository
2. Copy the `mod` folder contents to your Victoria 3 mods directory:
   - Windows: `Documents\Paradox Interactive\Victoria 3\mod\`
   - Linux: `~/.local/share/Paradox Interactive/Victoria 3/mod/`
3. Enable the mod in the Paradox Launcher

## Compatibility

- Game Version: 1.8.*
- Multiplayer: Yes (synchronized)

## Technical Details

This mod modifies the `buildable_by_companytype` property for each grain farm building type, adding `company_basic_food_industry` to the list of companies that can build these farms through Industry Rights Charters.
